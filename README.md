# Zomato Customer Churn Analytics

An interactive customer churn prediction system built using Streamlit.

## Problem Statement
Predict customers likely to churn based on their behavior and help businesses take proactive retention actions.

## Features
- Churn prediction with probabilities
- Interactive dashboard
- Threshold-based risk control
- Customer-level insights
- CSV download
- Single customer simulation

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib, Seaborn
